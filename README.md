# Ferran Moreno — Web 4.0 Final

## Qué incluye
- Home principal
- Página separada `/wall-of-trust`
- SEO base (metadata, robots, sitemap, schema Person)
- Links de contacto preparados
- Testimonios preservados sin reescritura

## Deploy en Vercel
1. Sube esta carpeta a GitHub o arrástrala a Vercel.
2. Framework: **Next.js**
3. Build command: `npm run build`
4. Output: automático

## Desarrollo local
```bash
npm install
npm run dev
```
